# Installing

Clone this project and build it:

```bash
$ git clone git@github.com:nawarian/php-lexer.rs.git
$ cd php-lexer.rs
$ cargo build
```

# Running tests

Just use the cargo tool for such:

```bash
$ cargo test
```
